# Workflow: Research Lifecycle

Use when a `W-xxx` research project or research lifecycle stage is explicitly assigned.

This workflow is a pointer, not a new process.

Required route:

1. Confirm task type `T10_RESEARCH`.
2. Read `06_research_program/RESEARCH_LIFECYCLE.md`.
3. Read the active or completed project folder for the named `W-xxx`.
4. Use the stage-specific template in `06_research_program/templates/`.
5. Do not integrate into `03_knowledge_base/` before an Editor Decision.

- `TASK_TYPES.md`
