# Workflow: Book Mode

Use when a book or primary source has been explicitly approved for processing.

This workflow is a pointer, not a new process.

Required route:

1. Confirm task type `T1_BUCHANALYSE`.
2. Read the files allowed by `TASK_TYPES.md`.
3. Follow `00_project/SALES_CODEX_OPERATING_MANUAL.md`.
4. Execute the pipeline in order:

   SRC -> ST -> A -> MEC -> P -> T -> MOD -> VAL -> BOOK_REVIEW

Stop only for the stop conditions in the Operating Manual and the active task instructions.

Primary references:

- `TASK_TYPES.md`
- `00_project/SALES_CODEX_OPERATING_MANUAL.md`
- `00_project/COWORK_EXECUTION_PROTOCOL.md`
