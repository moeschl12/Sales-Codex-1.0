# Principle Extraction Standard

## Zweck

Dieses Dokument definiert, wie aus Quellen belastbare Codex-Prinzipien gewonnen werden.

## Grundregel

> Wir extrahieren Erkenntnisse, nicht Inhalte.

## Sieben Extraktionsstufen

### 1. Rohaufnahme

Erfasse:
- Titel
- Autor
- Jahr
- Zielgruppe
- Kontext
- Hauptthese
- Quellenklasse

### 2. Zerlegung

Zerlege die Quelle in:
- Aussagen
- Behauptungen
- Modelle
- Methoden
- Definitionen
- Beispiele
- Gegenargumente

### 3. Klassifizierung

Ordne jedes Element ein als:
- Definition
- Beobachtung
- Empfehlung
- Methode
- Prinzip
- Hypothese
- Fallbeispiel
- Meinung

### 4. Prinzip-Extraktion

Frage:
Welche allgemeine Erkenntnis steckt hinter der konkreten Aussage?

### 5. Mechanismus-Identifikation

Frage:
Warum funktioniert dieses Prinzip?

Mögliche Mechanismen:
- Selbstüberzeugung
- Verlustaversion
- Vertrauen
- Commitment
- kognitive Dissonanz
- Reaktanz
- soziale Bewährtheit

### 6. Validierung

Vergleiche mit:
- anderen Autoren
- Forschung
- Praxisbeobachtung
- logischer Konsistenz

### 7. Integration

Entscheidung:
- neues Prinzip aufnehmen
- mit bestehendem Prinzip zusammenführen
- kontextuell spezialisieren
- ablehnen und begründen

## Reifegrad

Zusätzlich zum Evidenzlevel kann ein Prinzip einen Reifegrad erhalten:

- M1: neu entdeckt
- M2: mehrfach gestützt
- M3: gut verstanden
- M4: vollständig integriert
- M5: kanonisches Prinzip
