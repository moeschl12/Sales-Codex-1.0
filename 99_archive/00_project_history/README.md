# 00_project History Archive

This directory contains completed Sales Codex project reports that are no
longer part of the active control plane.

The first batch was moved from `00_project/` on 2026-07-13 after a repository-
wide reference audit. The original logical groups are preserved where useful.

Archived material is evidence and history. It must not be loaded during a
normal session and it must not be treated as an active launcher, decision
register, or current quality register.

Before moving another file here:

1. Check active-control inbound references.
2. Update all repository references.
3. Preserve the complete evidence bundle.
4. Re-run the link and startup-path checks.
