# MOD-0005 — Gap Selling Modell

## Modell-ID

MOD-0005

## Name

Gap Selling Modell (Keenan, 2018)

## Quelle

SRC-0005 — Gap Selling, Keenan, 2018

## Kernthese

Kaufentscheidungen entstehen ausschließlich aus einem quantifizierbaren Gap zwischen Current State (Schmerz, Kosten, Probleme) und Future State (angestrebter Zustand). Die Aufgabe des Verkäufers ist Diagno-stiker, nicht Überredungskünstler: erst vollständig verstehen, dann punktgenau lösen.

**Kern-Formel:** Future State – Current State = The Gap → je größer der Gap, desto stärker die Kaufmotivation

## Kausallogische Struktur

```
[Vor-Gespräch]
     │
     ▼
 PIC erstellen (T-0024)
 Branchenrecherche → Hypothesen über Probleme / Root Causes / Future State
     │
     ▼
[Phase 1: CREDIBILITY AUFBAUEN]
 A-0029: Expertise als Gate
 MEC-0008: Autorität-Wahrnehmung durch Branchenkenntnis + PIC-Antizipation
     │
     ▼
[Phase 2: DISCOVERY — Current State erschließen]
 T-0023: Gap Discovery Questioning (Probing → Process → Provoking → Validating)
 ST-0134: 5 Elemente des Current State (Fakten / Techn. Probleme / Business Probleme / Root Causes / Emotionaler Zustand)
 Quantifizierungsregel: Keine offenen Antworten akzeptieren
     │
     ▼
[Phase 3: FUTURE STATE & THE WHY erschließen]
 ST-0136: Future State als Veränderungsmagnet
 A-0028: Intrinsische Motivation (The Why) — emotionaler Kern
 MEC-0002: Verlustaversion aktiviert (Gap = Kosten des Status Quo)
     │
     ▼
[Phase 4: GAP QUALIFIZIEREN]
 Gap-Formel: Future State – Current State = The Gap
 A-0025: Budget ist Funktion der Gap-Größe
 A-0027: COI > Investitionskosten → Kauf rational
 Falls Gap zu klein oder Fit nicht vorhanden → Exit (vgl. T-0021)
     │
     ▼
[Phase 5: ENTSCHEIDUNGSKRITERIEN & KAUFPROZESS]
 T-0025: Inkonsistenz-Challenge ("Ich bin verwirrt, Sie sagten...")
 A-0022: Konsensbasierter Kauf — alle Entscheider müssen berücksichtigt werden
 MEC-0014: Konsens als Kaufsicherheit
     │
     ▼
[Phase 6: DEMO / LÖSUNG]
 T-0022: Gap-Demo-Methode (PIC → Demo Map → max. 6 Features → "when"-Sprache)
 MEC-0015: Kognitive Überlastung vermeiden
 MEC-0004: Demo knüpft an Discovery-Commitments an
     │
     ▼
[Gesamter Prozess: YES-SEQUENZ]
 ST-0145: Jedes Ja = Mikro-Sale; Gesamtkauf = Sequenz von Mikro-Commitments
 MEC-0004: Commitment/Konsistenz-Kette
     │
     ▼
[Abschluss]
 "Facilitation of Change" — nicht Produktverkauf
 A-0024: Gap war Kaufvoraussetzung ✓
```

## Modell-Vergleiche zu anderen Sprint-Büchern

### vs. SPIN Selling (MOD-0001)

| Dimension | SPIN Selling | Gap Selling |
|---|---|---|
| Kerntool | Fragetechnik (SPIN) | Fragetechnik (Probing/Provoking) + PIC |
| Was erschlossen wird | Implizite → Explizite Bedarfe | Current State (5 Elemente) + Future State + Why |
| Lösungstiming | Nach Explizit-Bedarf-Formulierung | Nach vollständiger Gap-Diagnose |
| Quantifizierungsfokus | Implication Questions (Kosten) | Alle Fragen müssen quantifiziert werden |
| Ähnlichkeit | Hoch — strukturelle Übereinstimmung in Fragelogik | |

### vs. Challenger Sale (MOD-0004)

| Dimension | Challenger Sale | Gap Selling |
|---|---|---|
| Einstieg | Teach (proaktiver Insight) → Tailor → Take Control | Discover (Diagnose-First) → Gap → Lösung |
| Credibility-Erzeugung | Durch Insight-Disruption (MEC-0013) | Durch Diagnose-Tiefe (Root Cause-Kenntnis) |
| Käuferrolle | Passiv rezipierend (wird gelehrt) | Aktiv explorierend (wird befragt) |
| Widerspruch | A-0020 (Kunden wollen gelehrt werden) vs. P-0025 (Fragen First) | UNGELÖST → S1-SYNTHESIS |

### vs. Never Split the Difference (B-0003)

| Dimension | Never Split | Gap Selling |
|---|---|---|
| Emotional Layer | Tactical Empathy, Labeling | "The Why" — direkte Abfrage |
| Methode | Implizite Emotions-Erschließung | Explizite Motivations-Fragen |
| Überlappung | Beide erschließen emotionale Hinterbühne | Unterschiedliche Gesprächsführung |

### vs. Influence (MOD nicht erstellt)

| Cialdini-Prinzip | Gap Selling Entsprechung |
|---|---|
| Commitment/Consistency (MEC-0004) | Yes-Sequenz als Mikro-Commitment-Kette |
| Authority (MEC-0008) | Credibility durch Diagnose-Tiefe (A-0029) |
| Liking (MEC-0007) | Explizit abgelehnt: A-0029 > Sympathie |
