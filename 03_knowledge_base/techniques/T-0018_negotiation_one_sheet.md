# T-0018 — Negotiation One Sheet (Vorbereitungsrahmen)

## Technique ID

T-0018

## Source ID

SRC-0003

## Name

Negotiation One Sheet — Fünf-Sektionen-Vorbereitungsrahmen

## Definition

Das Negotiation One Sheet ist ein strukturiertes Vorbereitungsformat, das alle wesentlichen Elemente einer Verhandlung auf einer Seite zusammenfasst. Es erzwingt aspirationale Zielformulierung (statt BATNA-Denken), empathiebasierte Zusammenfassung der Gegenüber-Perspektive und die Vorausplanung aller wichtigen Verhandlungstechniken (Labels, Fragen, Angebote). Es ist kein Skript, sondern eine Denk-Struktur.

Die fünf Sektionen (ST-0106):
1. **Goal:** Bestes realistisch erreichbares Ergebnis (aspirational, nicht BATNA)
2. **Summary:** 2-Satz-Zusammenfassung der Gegenüber-Perspektive (zielt auf "That's right"-Moment)
3. **Labels / Akkusationsaudit:** 3–5 vorgeplante Labels + 3–5 vorweggenommene Negativ-Annahmen des Gegenübers
4. **Kalibrierte Fragen:** 3–5 What/How-Fragen für verschiedene Gesprächssituationen
5. **Noncash-Angebote:** Was kann man geben, das kein Geld kostet aber Wert für das Gegenüber hat?

## Zweck

- Aspirationales Ziel statt Defensiv-Ziel (BATNA-Falle vermeiden: "you fall to your highest level of preparation")
- Verhandlung strukturieren ohne sie zu skripten
- Empathie-First als strukturellen Zwang einbauen (Sektionen 2+3 kommen vor taktischen Elementen)
- Black-Swan-Raum lassen: One Sheet ist Ausgangspunkt, nicht Endpunkt

## Angewendetes Prinzip

- P-0016 — Tactical Empathy: Sektionen 2+3 erzwingen Perspektivübernahme vor der Taktik
- P-0020 — Black-Swan-Orientierung: One Sheet als Hypothesenrahmen, nicht als Positions-Plan
- P-0017 — Autonomie-Illusion: Kalibrierte Fragen (Sektion 4) müssen vorab formuliert sein, um im Gespräch fließend einsetzbar zu sein

## Genutzter Mechanismus

Kein direkter psychologischer Mechanismus — One Sheet ist ein Vorbereitungswerkzeug, das sicherstellt, dass andere Techniken (T-0012 bis T-0017) im Gespräch verfügbar sind.

## Gesprächszustand

Pre-Contact

## Beispiel-Formulierung

**One Sheet Struktur (ausgefüllt für Gehaltsverhandlung):**

```
GOAL
"Ich verlasse das Gespräch mit einer Zusage von $X + Entwicklungsbudget $Y."
(Nicht: "Ich will mindestens $Z.")

SUMMARY (Counterpart-Perspektive, Ziel: "That's Right")
"Es klingt so, als hätten Sie ein enges Budget dieses Quartal 
und müssten jeden neuen Kostenpunkt intern rechtfertigen."

LABELS / AKKUSATIONSAUDIT
- "It seems like budget is a real constraint right now."
- "It looks like you might be wondering if the role justifies this level."
- "It sounds like timing might be a problem."

KALIBRIERTE FRAGEN
- "What would need to happen for this to work?"
- "How do you see this fitting into the team's goals?"
- "What does success in this role look like in Year 1?"

NONCASH-ANGEBOTE
- Flexible Start-Datum
- Remote-Work-Tage
- Übernahme von Weiterbildungskosten durch Eigenleistung
```

## Risiken

- One Sheet ≠ Skript: Wenn es wie ein Skript benutzt wird, verliert die Verhandlung Authentizität
- BATNA-Denken durch Hintertür: Wenn Goal zu defensiv formuliert wird, verfehlt es den Zweck
- Sektionen 4+5 werden überpriorisiert: Viele Praktiker fokussieren auf Fragen und Angebote, überspringen aber Sektionen 2+3 (Summary + Empathie)

## Typische Fehlanwendung

1. **Ziel = BATNA:** "Ich will mindestens $X" → Mindest-Denken statt Best-Case-Denken
2. **Leere Summary:** Zwei Sätze, die das Gegenüber gar nicht erfassen → kein "That's Right"-Moment möglich
3. **5 Labels als Checkliste abarbeiten:** Labels müssen im Gespräch situativ eingesetzt werden, nicht der Reihe nach

## Ethisches Risiko

Niedrig — Vorbereitung ist ethisch unbedenklich. One Sheet macht Verhandler kompetenter, nicht manipulativer.

## Evidenzlevel

E2 — Voss FBI-Praxis und Consulting-Erfahrungen. "7:1 ROI auf Vorbereitungszeit" (ST-0106) ist unbelegte Voss-Behauptung. Goal-Setting-Forschung (Locke & Latham, 1990: spezifische, ehrgeizige Ziele → bessere Performance) stützt die aspirationale Zielformulierung.

## Verknüpfte Kompetenzen

Verhandlungsvorbereitung, Perspektivübernahme, Stakeholder-Analyse
