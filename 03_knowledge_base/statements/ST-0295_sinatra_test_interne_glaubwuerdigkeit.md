# ST-0295 — Sinatra-Test: Ein einziges passendes Beispiel kann umfassende Glaubwuerdigkeit erzeugen

## Statement ID

ST-0295

## Source ID

SRC-0015

## Quelle

SRC-0015, Kapitel 4 "Credible"

## Originalaussage

"An example passes the Sinatra Test when one example alone is enough to establish credibility in a given domain... It's the Sinatra Test: If you can make it there, you can make it anywhere."

## Paraphrase

Bestimmte Beispiele/Referenzen sind so anspruchsvoll oder prestigetraechtig, dass ihre erfolgreiche Bewaeltigung allein ausreicht, um Kompetenz in einem gesamten Feld zu belegen (z.B. eine Logistikfirma, die termingerecht ein hochsensibles Harry-Potter-Buch-Release auslieferte, muss ihre Zuverlaessigkeit nicht durch weitere Statistiken beweisen). Dies ist eine von drei im Buch beschriebenen Quellen "interner" Glaubwuerdigkeit (neben konkreten Details und Statistiken auf menschlichem Massstab), die OHNE externe Autoritaet auskommt.

## Aussage-Typ

Beobachtung/Konzept (Fallbeispiel-basiert, keine kontrollierte Studie — Praktiker-Heuristik)

## Kontext

Kap. 4, Teil der Analyse verschiedener Glaubwuerdigkeits-Strategien (externe Autoritaet/Antiautoritaet, interne Details/Statistiken, Sinatra-Test, testbare Nachweise). Ergaenzt die im Codex bereits vorhandene Cialdini-Autoritaets-Mechanik (MEC-0005-0009) um eine NICHT-autoritaetsbasierte Glaubwuerdigkeitsquelle.

## Mögliche Annahmen

Kein eigener MEC noetig — dies ist eine Anwendungsheuristik der Verfuegbarkeitsheuristik/Reprasentativitaetsheuristik (bereits durch MOD-0010 Kahneman-Bias-Karte indirekt abgedeckt), aber die konkrete Vertriebsanwendung ist neu. Formalisiert als A-0060 (Extrembeispiel generalisiert Kompetenzeindruck).

## Mögliche Prinzipien

P-0052 (neu, Glaubwuerdigkeit durch interne Quellen statt externe Autoritaet).

## Offene Fragen

Keine kontrollierte experimentelle Pruefung des "Sinatra Test"-Konzepts selbst identifiziert — es handelt sich um eine von den Autoren benannte, aber nicht selbst experimentell getestete Heuristik.
