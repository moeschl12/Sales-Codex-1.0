# ST-0242 — MIT-Honor-Code-Signatur eliminiert Betrugsverhalten (verwandter Befund zu ST-0241)

## Statement ID

ST-0242

## Quelle

SRC-0012 (Predictably Irrational, Ariely 2008/2009), Kapitel 11 "The Context of Our Character, Part I"

## Originalaussage

"participants pocketed their answer sheets, but had also signed the honor code statement. In this case they claimed to have solved, on average, three problems—exactly the same number as the control group. [...] The effect of signing a statement about an honor code is particularly amazing when we take into account that MIT doesn't even have an honor code."

## Paraphrase

In einer Folgestudie zum selben Forschungskomplex wie ST-0241 (Mazar/Amir/Ariely-Linie) sollten Probanden vor einer betrugsanfälligen Aufgabe eine Erklärung unterschreiben ("Ich verstehe, dass diese Studie dem MIT Ehrenkodex unterliegt") — obwohl MIT zum Studienzeitpunkt keinen solchen Kodex hatte. Die Unterschriften-Gruppe betrog im Originalbefund nicht (identisch zur betrugsfreien Kontrollgruppe).
