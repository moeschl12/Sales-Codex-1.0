# Setup

## 1. Workspace öffnen

Den Ordner `sales-codex-workspace-v1.0` in VS Code öffnen.

## 2. Git lokal initialisieren

Im VS-Code-Terminal ausführen:

```bash
git init
git add .
git commit -m "Initial Sales Codex Workspace v1.0"
```

## 3. Optional: GitHub später verbinden

Erst wenn gewünscht:

```bash
git remote add origin <REPOSITORY_URL>
git branch -M main
git push -u origin main
```
